## Project Overview

This project is a real-time AI powered customer support assistant.
A user interacts with a chat interface on a website and receives automated responses generated by a Large Language Model (Google Gemini).

The system is not a simple chatbot UI — it is a complete end-to-end client → server → automation → AI pipeline deployed on the cloud.

The application demonstrates how modern applications integrate frontend interfaces with backend workflows and AI services using webhooks and APIs.

---

##  (Architecture)

1. User sends a message from the website chat interface.
2. The frontend sends a POST request to a serverless API endpoint.
3. The API forwards the request to an n8n webhook.
4. n8n triggers a workflow automation.
5. The workflow sends the message to Google Gemini (LLM).
6. Gemini generates a response.
7. n8n returns the response as JSON.
8. The website displays the reply in the chat window.

---

## System Architecture

**Frontend (Client)**

* HTML
* CSS
* JavaScript
* Hosted on Vercel

**Middleware (Serverless API)**

* Vercel Serverless Function (`/api/chat`)
* Acts as a secure proxy between browser and backend

**Backend Automation**

* n8n Cloud (Webhook triggered workflow engine)

**AI Engine**

* Google Gemini API (Large Language Model)

Browser → Vercel API → n8n Webhook → Gemini API → n8n → Vercel → Browser
